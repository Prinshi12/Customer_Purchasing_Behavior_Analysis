# Customer Purchasing Behavior Analysis

## Overview
Analysis of customer purchasing behavior to identify key drivers of spending, loyalty, and repeat purchases using Python, SQL, and Power BI.



## Objective
To understand how demographics, discounts, reviews, shipping methods, and subscriptions influence customer purchasing decisions.


## Dataset
- 3,900 rows | 18 columns  
- Customer demographics, purchase details, and shopping behavior  
- Missing values handled in Review Rating column  



## Tools & Technologies
- Python (Pandas, NumPy)
- MySQL
- Power BI
- Excel


## Methodology
- Data cleaning and feature engineering using Python  
- Business analysis using SQL  
- Visualization and insights using Power BI  



## Key Insights
- Female customers generate higher revenue despite smaller population  
- Loyal customers contribute ~89% of total revenue  
- Express shipping users show higher average spend  
- Only 27% customers are subscribers, indicating growth opportunity  



## Business Recommendations
- Strengthen customer loyalty programs  
- Promote subscription benefits  
- Optimize discount strategies  
- Focus marketing on high-revenue customer segments  



## Conclusion
This project demonstrates how data analysis can generate actionable insights to improve revenue, customer loyalty, and marketing effectiveness.
